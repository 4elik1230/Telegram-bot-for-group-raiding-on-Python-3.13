# Telegram-bot-for-group-raiding-on-Python-3.13
+ Это Telegram бот для рейда групп на Python.
+ Этот бот умеет управлять группой удалено.
+ Бот сделан на версии Python 3.13
[Установить Python 3.13](https://www.python.org/downloads/release/python-3130/)
___

# Как сделать чтобы он управлял группой?
+ Надо добавить в группу в которой нам надо управлять.
+ Сделать его администратором. 
+ Теперь напишите в личные сообщения боту команду /start.
+ После того когда вы введите эту команду, появится кнопка "добавить группу".
+ Вы нажимаете на "добавить группу"
+ Вы должны написать название группы, можете написать любое название, даже если это не название группы.
+ Напишите id группы.
+ И напишите id пользователей группы.
+ Заново нажимайте на /start.
+ И у вас появится кнопка с названием группы.
+ Нажимайте на него

# Команды бота
1. Команды для всех.
+ /mute - мут, чтобы замутить пользователя надо ответить на сообщение человека которого хотите замутить. Еще надо написать время мута, например: /mute 5 (5 это 5 часов).
+ /ban - бан, чтобы замутить пользователя надо ответить на сообщение человека которого хотите забанить.
+ /rules - правила группы.
+ /setrules - установить правила.
+ /help - выдает все команды для всех.
2. Команды для вас.
+ /deleteallusers - удалить всех пользователей (эта команда удаляет только обычных пользователей, а админов нет из-за правил Telegram).
+ /forwardon - включить пересылку сообщений.
+ /forwardoff - выключить пересылку сообщений.
+ /getlink - ссылка на группу.
+ /deleteconfig - удалить кнопку с группой если уже не пользуйтесь ей..
