# Telegram-bot-for-group-raiding-on-Python-3.13
Это Telegram бот для рейда групп на Python 3.13.
Этот бот умеет управлять группой удалено.
___
# что надо сделать перед запуском бота?
+ Ввести токен своего бота
2. Ввести свой id, это можно сделать на этом боте - @UserInfeBot
___
# Как сделать чтобы он управлял группой?
1. Надо добавить в группу в которой нам надо управлять.
2. Сделать его администратором. 
3. Теперь напишите в личные сообщения боту команду /start.
4. После того когда вы введите эту команду, появится кнопка "добавить группу".
5. Вы нажимаете на "добавить группу"
6. Вы должны написать название группы, можете написать любое название, даже если это не название группы.
7. Напишите id группы.
8. И напишите id пользователей группы.
9.  Заново нажимайте на /start.
10.  И у вас появится кнопка с названием группы.
11.  Нажимайте на него
И все вы можете управлять группой
